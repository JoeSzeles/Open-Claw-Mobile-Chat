Quick Start

  1. Pair phone to car via Bluetooth
  2. Open this Mobile Chat page in Chrome
  3. Tap Drive or press Play on radio to start hands-free
  4. Speak naturally — 2 second pause sends your message
  5. Agent responds through car speakers, mic stays off while speaking
  6. Radio display shows what's happening: listening, waiting, speaking
  7. Press Next to send message, Prev to toggle mic
